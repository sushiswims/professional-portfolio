# Portfolio Website

## Project Description
This is a personal portfolio websiteIt showcases my work, skills, and projects in PR. The site is designed to be clean, modern, and easy to navigate.

## Features
- Responsive homepage layout
- About section with personal information
- Project / case study showcase
- Contact section
- Clean and consistent design
- Organized file structure

## Setup Instructions
1. Download or clone the repository  
2. Open the project folder  
3. Open `index.html` in your browser (double-click or right-click → open with browser)

No installations are required.

- index.html
